## ⬛ Tentang Boemi

**Boemi Hair Studio** adalah website *landing page* responsif bernuansa gelap (*dark-mode*) yang dirancang untuk memberikan kesan profesional, premium, dan elegan. Dibangun khusus untuk kebutuhan *barbershop* atau *hair studio* modern yang mengutamakan kualitas visual.

## ⬜ Fitur Utama

- **Desain Premium**: Tema hitam-putih klasik dengan aksen minimalis yang menonjol.
- **Responsif Penuh**: Tampilan optimal di semua layar (Mobile, Tablet, Desktop).
- **Galeri Dinamis**: Layout grid asimetris yang menawan, mendukung elemen gambar dan video.
- **Katalog Layanan**: Daftar harga dan layanan terstruktur yang sangat mudah dibaca.
- **Performa Ringan**: Dibangun murni menggunakan Native HTML, CSS, dan Vanilla JS (Tanpa *framework* berat).

---

## ⬛ Teknologi

- **HTML5** – Struktur halaman dan semantik.
- **CSS3** – *Flexbox*, *Grid Layout*, *Media Queries*, dan *Micro-animations*.
- **Vanilla JavaScript** – Logika untuk Menu Mobile (Hamburger Menu).
- **Google Fonts** – Paduan font klasik *Cormorant Garamond* (Judul) & modern *Outfit* (Teks).

## ⬜ Struktur Direktori

```text
📁 Boemi Hair Studio
├── 📄 index.html      # Struktur utama halaman
├── 📁 css
│   └── 🎨 style.css   # Aturan styling dan responsivitas
├── 📁 js
│   └── 📜 script.js   # Logika interaksi DOM
└── 📁 assets          # Direktori gambar dan video (Hero, Galeri)
```

---

## ⬛ Penggunaan & Modifikasi

1. **Menjalankan Web**: Buka file `index.html` langsung menggunakan browser (Chrome, Safari, dll). Tidak memerlukan server khusus.
2. **Mengganti Media**: Untuk mengubah gambar atau video, pastikan Anda mengganti file di dalam folder `assets/` dan menyesuaikan nama file/ekstensinya (misal: `.jpg`, `.jpeg`, `.mp4`) di dalam `index.html`.
